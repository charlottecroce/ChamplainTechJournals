
## create new PSModule
cd to modules folder
```
New-ModuleManifest -Path .\480-utils.psd1 -RootModule 480-utils.psm1
```

## add new PSModule to PSModulePath
code $profile
```
$env:PSModulePath = $env:PSModulePath + ':/home/charlotte/ChamplainTechJournals/adv-topics-sec480/modules'
```
restart vscode. modules folder should be in PSModulePath now

## to import or update PSModule

cd to class root
```
Import-Module '480-utils' -Force
```
