## setup VPN

champlain
```
# create access list for traffic allowed through tunnel
access-list 110 permit ip 172.16.84.0 0.0.0.255 192.168.25.0 0.0.0.255

# Configure IKE Phase 1 ISAKMP Policy
crypto isakmp policy 10
encryption aes 256
authentication pre-share
group 5
exit
crypto isakmp key NET330 address 140.230.18.1

# Configure the IKE Phase 2 IPsec policy
crypto ipsec transform-set VPN-SET esp-aes esp-sha-hmac
crypto map VPN-MAP 10 ipsec-isakmp
description VPN connection to Middlebury
set peer 140.230.18.1
set transform-set VPN-SET
match address 110
exit

# Configure the "crypto map" on the outgoing interface.
interface f0/0
crypto map VPN-MAP
```
