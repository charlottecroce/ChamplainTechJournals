### 1. Configure Hospital Router
### vlan configuration
Hospital Router
```
ip routing
interface vlan 1
no shutdown
vlan 2
name Counseling
vlan 3
name Clinic
vlan 4
name Visitor
vlan 5
name Office
```
### Configure trunk ports to connect to core switches
On hospital router
```
# trunk port
interface range f0/2-3
switchport trunk encapsulation dot1q
switchport mode trunk
switchport trunk allowed vlan 1,2,3,4,5
```

### vlan ip addresses
```
interface vlan 1
ip address 10.16.0.1 255.255.255.0
interface vlan 2
ip address 10.16.1.1 255.255.255.0
interface vlan 3
ip address 10.16.2.1 255.255.254.0
interface vlan 4
ip address 10.16.4.1 255.255.254.0
interface vlan 5
ip address 10.16.6.1 255.255.254.0
```

## 2. Configure North-Core & South-Core Switches
### vlans configuration
```
interface vlan 1
no shutdown
vlan 2
name Counseling
vlan 3
name Clinic
vlan 4
name Visitor
vlan 5
name Office
```

Trunk port to border
```
interface f0/2
switchport trunk encapsulation dot1q
switchport mode trunk
switchport trunk allowed vlan 1,2,3,4,5
```
Trunk port to edge
```
interface f0/1
switchport trunk encapsulation dot1q
switchport mode trunk
switchport trunk allowed vlan 1,2,3,4,5
```


### 3. Configure North Wing Edge Switch (can copy to South later)
trunk to core
```
# trunk port
interface f0/1
switchport trunk encapsulation dot1q
switchport mode trunk
switchport trunk allowed vlan 1,2,3,4,5
```

access ports
```
interface range fa0/2-7
switchport access vlan 3
interface range fa0/8-11
switchport access vlan 4
interface range fa0/12-17
switchport access vlan 5
```


#### counseling switch
```
interface range fa0/2-10
switchport access vlan 2
```
