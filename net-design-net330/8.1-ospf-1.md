## configure ospf

West
```
router ospf 1
network 10.16.40.0 0.0.0.255 area 0
network 10.16.1.32 3.255.255.255 area 0
```

East
```
router ospf 1
network 10.16.20.0 0.0.0.255 area 0
network 10.16.1.20 3.255.255.255 area 0
```

DataCenter
```
router ospf 1
network 10.16.20.0 0.0.0.255 area 0
network 10.16.1.32 3.255.255.255 area 0
network 10.16.1.0 7.255.255.255 area 0
```

Border
```
router ospf 1
network 10.16.1.0 7.255.255.255 area 0
```

verify ospf routes (will be marked as 'O' in route table)
```
show ip route
```
