# Statement

Given a year as an integer, print "YES" if it's a part of the 21st century and print "NO" otherwise.

NOTE: The 21st Century starts with 2001 and ends with 2100 (2000 is not part of the 21st century).

# Example input #1

```
1790
```

# Example output #1

```
NO
```

# Example input #2

```
2037
```

# Example output #2

```
YES
```
